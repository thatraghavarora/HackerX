Demo folder

This demo provides an in-memory demo account with a $100 starting balance and simple functions to add incomes, add expenses, and delete transactions.

How to open:

1. Start the frontend (from `frontend`):

```bash
npm run dev
```

2. Open http://localhost:3000/demo to view the demo page.

Notes:
- This demo is in-memory only and does not interact with the smart contract or blockchain.
- Use it to preview UI and functionality without wallet connection.
